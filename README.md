# Credit Risk Analysis

## Project Overview
Analysis of 1,125 bank customers to identify credit risk patterns
using real financial data.

## Key Findings
- 20% of customers are high risk (225 out of 1,125)
- Risky customers have 2x higher overdue payments (369 vs 151)
- High balance customers are more likely to default

## Tools Used
- Python (Pandas, Matplotlib)
- Google Colab
- GitHub

- Untitled0.ipynb — Full analysis notebook
